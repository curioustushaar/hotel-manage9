# Sound Files

This directory should contain the sound effects used by the application.
The `SoundManager` expects the following files:

- `click.mp3` - Default click sound
- `typing.mp3` - Typing sound for inputs
- `notification.mp3` - Notification sound
- `success.mp3` - Success action sound
- `alert.mp3` - Error/Alert sound

Please add your sound files here for the sound functionality to work.
